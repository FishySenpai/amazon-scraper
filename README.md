# Amazon Product Scraper

This is a simple full-stack project using Bun (Express), JSDOM, and Axios on the backend, and Vite with vanilla JavaScript on the frontend.

## Getting Started

### Backend

```bash
cd backend
bun install
bun index.ts
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Visit the frontend at `http://localhost:5173`
